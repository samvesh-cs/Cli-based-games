# 🎯 Guess The Number Game (Python CLI)

A simple command-line **Guess the Number game** written in Python.  
The computer randomly selects a number within a user-defined range, and the player must guess it.

The program guides the player with hints like **"Higher Number Please"** or **"Lower Number Please"** until the correct number is guessed.

---

## 🚀 Features

- User chooses the number range
- Random number generation using Python's `random` module
- Input validation to prevent invalid entries
- Hints to guide the player
- Tracks:
  - Total attempts
  - Valid guesses
- Option to restart the game

---

## 🛠️ Technologies Used

- Python
- Random module
- Command Line Interface (CLI)

---

## ▶️ How to Run

1. Make sure Python is installed.

2. Clone the repository:
```
git clone https://github.com/samvesh-cs/Cli-based-games.git
```
3. Navigate into the folder

```
cd Cli-based-games
```

4. Run the program

```
python main.py
```
---

## 🎮 How the Game Works

1. Player enters a **lower limit** and **higher limit**.
2. The program selects a **random number within that range**.
3. Player guesses the number.
4. The program responds with hints:
   - "Higher Number Please"
   - "Lower Number Please"
5. The game continues until the correct number is guessed.
